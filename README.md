# chens_jujube

## Features
- 以蜜棗晨品牌為開發構想，以Vue 2開發純前端購物網站。
- 產品資料儲存Vuex，以操縱資料。
- 網頁内容大部分以手刻爲主，以sass輔助開發。
- 實作購物車功能，以localStorage儲存購物車資料。
- 連接 EmailJS API 以達成用戶與商家的聯係。
